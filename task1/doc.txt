Applikator #1

#1
Реализовать функцию isEqual(obj1, obj2) которая возвращает true если два объекта равны между собой и false - если не равны. Два объекта равны между собой если значения их полей равны между собой (обратите внимание, объект может быть вложен в объект и т.д.). Вы можете быть уверены что obj1 и obj2 являются полноценными объектами
Примеры:

isEqual ({ a: 2 }, { a: 2 }); // --> true
isEqual ({ a: 2 }, { a: "2" }); // --> false
isEqual ({ a: 2, b: [1, 2] }, { a: 2, b: [1, 2] }); // --> true
isEqual ({ a: 2, b: { k: [1, 2] } }, { a: 2, k: [1, 2] }); // --> false
 
#2
Реализовать функцию clone(obj1) которая возвращает копию объекта с абсолютно теми же полями. Копия такова, что никакие модификации ее или вложенных полей НЕ могут изменить объект obj2
Примеры (а какие здесь могут быть примеры):
 
isEqual (obj1, clone(obj1)); // --> true
 
Формат сдачи: заливаете ваш код на jsfiddle.com и в письме присылаете ссылку
Тема письма должна быть "#ApJS1 Фамилия Имя". В письме - ссылки на одно или два задания (как успеете)
